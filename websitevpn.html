 How to install

VPN Type is L2TP

For the hostname(server), type “customvpn.viewdns.net”

Don't worry about the 2nd line. It should be auto-filled

Find Shared Secret key and type “VPNSecretKey”

Then go down, find Username(account), type "servermac"

For the password, type _DATA_CONST

Don't worry about group name.


Tick the checkmark so the password is saved and you can connect seamlessly. Then click connect.



Troubleshooting

If any of your websites don't load, disconnect and reconnect.

If that doesn't fix it, just restart your pc.  If that doesn't resolve it, keep trying, the server slots may be full.
