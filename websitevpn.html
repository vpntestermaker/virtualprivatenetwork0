 How to install

VPN Type is L2TP

For the hostname(server), type “customvpn.viewdns.net”

Don't worry about the 2nd line.

Find Shared Secret key and type “VPNSecretKey”

Then go down, find Username(account), type "servermac"

For the password, type _DATA_CONST

Don't worry about group name.


Check the check so the password is saved and you can connect seamlessly and then click confirm.


If any of your websites don't load, disconnect and reconnect. This shouldn't happen as often as we're on Beta 2 now.


Troubleshooting

If you run into any connectivity problems just restart your pc.  If that doesn't resolve it, keep trying, the server slots may be full.
